## Setup

Follow these steps to set up and run the project:

1. **Navigate to the project directory:**

```sh
cd ./agents
```

2. **Set up your environment variables:**

```sh
KEY= # the private key of the wallet
OPEN_AI_API_KEY= # sk-proj-...
```

3. **Install dependencies:**

```sh
yarn@10.8.3 install
```

4. **Run the project:**

```sh
yarn@10.8.3 dev
```

5. Enjoy!
